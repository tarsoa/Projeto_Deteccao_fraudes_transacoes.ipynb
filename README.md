# Projeto_Deteccao_fraudes_transacoes.ipynb
Este projeto aplica técnicas de Machine Learning para detecção de fraudes em transações financeiras utilizando classificação supervisionada em um dataset altamente desbalanceado.  O objetivo é identificar transações fraudulentas com alta taxa de recall, reduzindo falsos negativos e melhorando a capacidade de prevenção de perdas financeiras.
